4 Big Five Scale description

This is the Mini-IPIP Big Five measure by Donnellan et al. (2006). It is a brief 20-item measure, with four items per trait, two items per trait reverse-coded (RC).

4.1 Variable names for each factor

4.1.1 Extroversion

  • Q8_2 (RC)
  • Q9_3
  • Q10_4 (RC)
